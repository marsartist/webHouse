css选择器优先级

发表于 2019-12-24 分类于 css ， 选择器

1. 为什么会有优先级？

当不同的选择器中，有相同的样式作用于同一个标签的时候就会出现样式冲突，而选择器优先级就是来解决冲突的。

2. 那么优先级的规则是怎样的？

优先级由高到低：

  • !important 规则的优先级最大
  • 内联样式
  • 外联样式，根据选择器权值累加，最大者优先级最高
    • 权值情况
      • id选择器 100
      • class选择器 10
      • 标签选择器 1
  • 自定义样式优先于浏览器的默认样式
  • 就近原则，当权值相同的时候，越靠近元素的css的样式优先级越高
  • 当权值相同的时候，后声明的样式会覆盖前声明的样式

3. 假设有11个标签选择器符合构成一个选择器，那它的优先级的比的class选择器的优先级高吗

p>p>p>p>p>p>p>p>p>p>p (11个)  
.p

答案是否定的，因为不管有多少个标签选择器叠加，它的优先级都不会高于class选择器，同理不论有多少个class选择器叠加，其优先级都不会高于id选择器
，而且叠加的选择器其实是有个数限制的
