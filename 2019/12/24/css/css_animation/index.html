css动画实现方式

发表于 2019-12-24 分类于 css ， 动画

1. transition

描述：将元素的某一个属性从“一个属性值”在指定的时间内平滑地过渡到“另外一个属性值”来实现动画效果
(注意是属性值)
属性：transition-property,transtion-duration,transition-timing-function,transition-delay

2. transform + transition

描述：transform属性是一些已经定义的动画，用来实现文字或图像的的各种变形效果，如位移、缩放、旋转、倾斜等。
通常结合transition或者animation来实现动画的效果
属性方法：
translate()位移
scale()缩放
rorate()旋转
skew()倾斜
tranform-origin(中心原点)

transform:translateX(-120px);
transition:-webkit-transform trandform,0.5s;

3. animate

描述：对于动画轨迹比较多的时候，用transition是实现不了，所以用animation来实现，
