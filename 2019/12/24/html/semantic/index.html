语义化

发表于 2019-12-24 分类于 html

1. 什么是语义化？

去掉用户自定义的样式后，页面仍呈现清晰结构（和浏览器的默认样式配合），比如<h>是粗体，字体大小是2em，（这里显示的是浏览器的默认样式）。即即使没有用户定义的样式，仍然知道清晰页面结构，比如标题。

2. 语义化解决了哪些问题？

  • 对于有视觉障碍的人阅读网站的时候，是通过阅读器。而阅读器实际上就是根据语义标签来识别文章的结构和内容。
  • pad、手机设备由于对css的支持比较弱，可能无法像浏览器一样渲染，而语义化的标签可以在一定的程度上的添加样式。
  • 语义化有利于seo，有助于seo网络蜘蛛爬取更多的信息，因为爬虫是根据标签来确定上下文和各个关键字的权重的。
  • 清晰的页面结构，有利于开发和维护。
